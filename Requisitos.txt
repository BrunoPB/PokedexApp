TELAS E FUNCIONALIDADES

LOGIN
	- Colocar nome de usuario e senha
	- Avisar se: 	o usuario for desconhecido
			algum campo não tiver sido preenchido
			a senha estiver errada
	- Botão que leva a tela de Register
	- Logar > Leva a tela de visualizar pokedex
	
REGISTER
	- Escolher nome de usuario
	- Escolher e confirmar senha
	- Avisar se: 	o nome de usuario ja esta sendo usado
			algum campo não estiver preenchido
			a senha não for confirmada corretamente
	- Botão de cancelar e levar de volta a tela de Login
	- Registrar dados no Banco de Dados > Leva a tela de Login
	
PERFIL
	- Alterar: 	nome de usuario
			senha
			pokemon de perfil
	- Botão para fazer Logout > Leva a tela de Login
	- Botão para voltar a tela de Visualizar Pokedex
	
VISUALIZAR POKEDEX
	- Mostrar todos os Pokemons adicionados

	- Ordenar pokemons
	- Botão que faz Logout > Leva a tela de Login
	- Botão que leva a tela de: 	Adicionar Pokemon
					Perfil
					Visualizar Pokemon (Ao clicar no pokemon)
